# 🧮 Python 计算器

一个基于 Tkinter 开发的简洁美观的桌面计算器应用——国防科大大一《大学计算机基础》课程作业。

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![Tkinter](https://img.shields.io/badge/Tkinter-GUI-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

## ✨ 功能特性

- 🎨 **美观界面** - 粉色主题，清新简约的设计风格
- ⚡ **快速计算** - 支持各种数学表达式的即时计算
- 🖥️ **桌面应用** - 基于 Tkinter 的原生桌面 GUI
- 🎯 **简单易用** - 直观的操作界面，无需学习成本

## 📸 界面预览

计算器界面包含以下元素：
- **输入区域** - 用于输入数学表达式
- **结果显示** - 清晰展示计算结果
- **计算按钮** - 一键触发计算操作
- **粉色主题** - 舒适的视觉体验

## 🚀 快速开始

### 环境要求

- Python 3.x
- Tkinter（通常随 Python 一起安装）

### 安装与运行

1. **克隆或下载项目**
   ```bash
   git clone <repository-url>
   cd pycalc_deek
   ```

2. **运行计算器**
   ```bash
   python calculator.py
   ```

3. **开始使用**
   - 在输入框中输入数学表达式
   - 点击"开始计算"按钮
   - 查看计算结果

## 💡 使用示例

### 基础运算

| 运算类型 | 示例输入 | 结果 |
|---------|---------|------|
| 加法 | `2 + 3` | 5 |
| 减法 | `10 - 4` | 6 |
| 乘法 | `6 * 7` | 42 |
| 除法 | `15 / 3` | 5.0 |

### 复杂表达式

```python
# 支持括号运算
(2 + 3) * 4  # 结果: 20

# 支持幂运算
2 ** 10  # 结果: 1024

# 支持混合运算
10 + 5 * 2 - 8 / 4  # 结果: 18.0
```

## 🎨 界面设计

- **窗口尺寸**: 800 x 110 像素
- **主题配色**: 粉色背景 (#d3fbfb)
- **字体**: 华文新魏 12号
- **布局**: 紧凑型设计，适合快速计算

## 📝 技术栈

- **GUI 框架**: Tkinter
- **编程语言**: Python
- **计算引擎**: Python 内置 eval() 函数

## ⚠️ 注意事项

- 计算器使用 `eval()` 函数进行计算，请确保输入的数学表达式合法
- 支持所有 Python 标准数学运算符和函数
- 窗口大小固定，不可调整

## 🔧 自定义配置

您可以轻松修改以下参数来自定义计算器：

```python
# 修改窗口标题
window.title('你的标题')

# 修改窗口尺寸
window.geometry('宽度x高度+x坐标+y坐标')

# 修改颜色主题
bg='#你的颜色代码'
fg='#你的文字颜色代码'
```
